# Gerador de Currículos (PHP)


Um projeto básico para gerar currículos usando PHP, HTML, CSS e jQuery.


## Como usar
1. Coloque a pasta em `C:/xampp/htdocs/gerador-curriculos`.
2. Crie `data/cvs/` com permissão de escrita.
3. Acesse [http://localhost/gerador-curriculos/public/index.php](http://localhost/gerador-curriculos/public/index.php).


## Funcionalidades
- Formulário de dados pessoais.
- Campos dinâmicos para experiências e referências.
- Cálculo automático de idade.
- Geração e visualização de currículo simples.
- Impressão ou PDF via navegador.
